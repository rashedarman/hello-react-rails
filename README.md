# 📖 Hello React Rails <a name="about-project"></a>

**Hello React Rails** is an exercise to setup RoR and React as one app with Webpack.

## 🗝️ Key Features <a name="key-features"></a>

- Implement webpack using `jsbundling-rails`
- Implement `redux-toolkit` for state management
- Implement `react-router` for client side routing

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- [Ruby](https://www.ruby-lang.org/en/documentation/installation/) 3.2.2+
- [Node.js](https://nodejs.org/) and a package manager such as [pnpm](https://pnpm.io/)

Optionally, in order to modify this project you should to have installed:

- Git.
- Code editor like: [Visual Studio code](https://code.visualstudio.com/), or others...

### Setup

Clone this repository to your desired folder:

```sh
  git clone git@github.com:rashedarman/hello-react-rails.git
```

### Usage

To run the project, execute the following commands:

```sh
  ./bin/dev
```
